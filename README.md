# MyGitRepository

Project Explanation:

Our group project consists of three main parts:
1. Grade Analyzer: calculates overall grades based on assignment weights, midterms, exams, and scores.
2. Calculator: a basic Python program to perform addition, subtraction, multiplication, and division.
3. Main: a main program to test and showcase the functionality of the grade analyzer and calculator programs.
